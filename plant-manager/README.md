## Sobre o projeto

O **PlantManager** é um aplicativo para facilitar a administração de suas plantas.

---

## Tecnologias utilizadas

Esse projeto foi desenvolvido utilizando as seguintes tecnologias abaixo:

- React Native
- Expo
- TypeScript
